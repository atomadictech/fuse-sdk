# Use Cases

## 1. Repository triage

Use `scan`, `search`, and `status` to understand a codebase before any synthesis step.

## 2. Agent runtime integration

Use `fuse-mcp` to expose the same workflow surface to an MCP-capable host without writing a custom adapter.

## 3. Verification-oriented automation

Use `verify_block`, `search_intent`, and `compose_stack` where downstream systems need explicit traceability.

## 4. Store maintenance

Use `lint_store`, `validate_store`, `deduplicate_store`, and `rebuild_indexes` in maintenance or CI workflows.

## 5. Language discovery and emit planning

Use `langs`, `polyglot`, `emit`, and `emit_corpus` to inspect available output modes and plan transformations.
